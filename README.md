# JavaScript 核心与进阶实践

## 简介

本项目收集了 JavaScript 语言中的核心知识点与进阶实践，包括但不限于异步编程、函数式编程、原型与继承、手写常用 API、Proxy、Symbol、发布订阅、CSS in JS 等内容，适合希望系统提升 JS 能力的开发者。

## 目录/主题

- **async.js**：Promise 异步编程基础
- **mini-promise.js**：手写 Promise 实现
- **generator.js**：Generator 及自动执行器
- **curry.js**：函数柯里化
- **compose.js / pipe.js**：函数组合
- **memorize.js**：函数记忆化
- **bind.js / call.js / apply.js**：手写 bind/call/apply
- **new.js**：手写 new 操作符
- **prototype.js**：原型与继承
- **proxy.js**：Proxy 与 Reflect 用法
- **pubsub.js**：发布订阅模式
- **sigotan.js**：单例模式
- **descontructor.js**：解构赋值
- **cssinjs.js**：CSS in JS 实践
- **testable.js**：简单的可测试函数与用例
- **arrow.md**：箭头函数与普通函数区别
- **symbol.md**：Symbol 基础与高级用法

## 特性

- 覆盖 JS 语言核心与高阶主题
- 代码均为可运行示例，便于理解
- 适合自学、面试复习、源码剖析

## 安装

```bash
git clone https://github.com/你的用户名/js-async.git
cd js-async
```

## 用法示例

可直接运行各 JS 文件或阅读 Markdown 文档：

```bash
node async.js
node mini-promise.js
# ...
```
